# Kurulum

## Gereksinimler
- Java 17+
- Paper/Spigot 1.19.4

## Adımlar
1. Derlenmiş JAR dosyasını `plugins/` klasörüne kopyalayın
2. Sunucuyu başlatın
3. `config.yml` oluşacaktır; düzenleyin
4. Gerekli izinleri verin (LuckPerms önerilir)

## Derleme (Geliştiriciler)
```
./gradlew build
```
Hedef JAR: `build/libs/`
